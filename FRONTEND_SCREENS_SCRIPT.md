## üìä Technical Highlights

### Frontend Stack
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **Animations**: Framer Motion
- **State Management**: React Context API
- **Web3**: ethers.js + MetaMask
- **i18n**: react-i18next with RTL support

### Backend Stack
- **Framework**: NestJS with TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Real-time**: Server-Sent Events (SSE)
- **Pub/Sub**: Redis
- **Storage**: Storacha (IPFS)
- **AI**: Groq API, Replicate API

### Agent Architecture
- **Orchestrator**: Coordinates workflow
- **Story Agent**: Groq AI for narratives
- **Asset Agent**: Replicate for images
- **Code Agent**: AI-generated games
- **Deploy Agent**: IPFS deployment

### Blockchain
- **Network**: Polygon (Mumbai testnet)
- **Token**: HHCW ERC-20
- **NFTs**: Achievement badges
- **Wallet**: MetaMask integration
